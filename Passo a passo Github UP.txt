1. Abrir o Gitbash na pasta com o repositório local do Git
2. Adicionar todos os arquivos para upar (git add . ou git add file_name)
3. Dar o commit para criar a (primeira ou) nova versão da aplicação
git commit -m "Nome da versão do programa"
4. Avisar que quero subir o meu código do Git local para o Github na branch main
git push -u origin main

git log --oneline - Permite visualizar todos os commits
